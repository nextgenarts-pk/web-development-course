# 📅 Week 1: Introduction to Web Development & HTML Basics

> **Full Stack Web Development Course — Integrated with AI** | [NextGen Arts](https://nextgenarts.pk)

---

## 🎯 Learning Objectives

By the end of this week, students will be able to:

- 🌐 Understand how the internet works (DNS, HTTP, browsers, servers)
- 🛠️ Set up a professional development environment (VS Code, Git, GitHub)
- 📝 Write well-structured HTML5 documents from scratch
- 🏗️ Use semantic HTML elements for accessible, meaningful web pages
- 📋 Build HTML forms, tables, and embed media elements
- 🚀 Publish a personal bio page on GitHub
