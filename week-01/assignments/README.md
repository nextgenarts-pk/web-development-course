## 🎯 Weekly Assignment: Personal Bio Page

Build a complete **Personal Bio Page** using only HTML5 (no CSS yet). This is your first real project!

---

## 📋 Requirements

### Must Have (Required):

- [ ] Valid HTML5 document structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- [ ] Page title and favicon
- [ ] Semantic HTML layout (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`)
- [ ] Navigation bar with at least 4 anchor links
- [ ] A heading with your full name
- [ ] A profile photo using `<figure>` and `<figcaption>`
- [ ] At least 2 paragraphs about yourself
- [ ] An unordered list of skills (minimum 5)
- [ ] An ordered list of goals or hobbies (minimum 3)
- [ ] An education/experience table with `<thead>`, `<tbody>`, `<tfoot>`
- [ ] A contact form with name, email, message fields, and a submit button
- [ ] Social media links in the footer
- [ ] Pushed to GitHub with at least 3 meaningful commits

---

## ⭐ Bonus Challenges

- [ ] Embed a YouTube video (your favorite tech talk)
- [ ] Add a Google Maps embed of your city
- [ ] Create a second page (`hobbies.html`) and link to it from the main page
- [ ] Use `<details>` and `<summary>` for an FAQ section
- [ ] Add `<audio>` element with a sample audio file
- [ ] Use HTML5 validation attributes on all form inputs

---

## 📤 Submission Guidelines

1. Create a GitHub repository named `personal-bio-page`
2. Push your completed project with meaningful commit messages
3. Share the repository URL with your instructor
4. Ensure the repository is **public**

---

## 📊 Grading Rubric

| Level | Criteria | Score |
|-------|----------|-------|
| 🟢 **Beginner** | Valid HTML5 structure, basic content, pushed to GitHub | 60-70% |
| 🟡 **Intermediate** | All required items complete, good semantic HTML, meaningful commits | 70-85% |
| 🔴 **Advanced** | All requirements + 3 or more bonus challenges, clean and well-organized code | 85-100% |
