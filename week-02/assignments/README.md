## 🎯 Weekly Assignment: Styled Bio Page

Transform your Week 1 **Personal Bio Page** from plain HTML into a **beautifully styled, professional webpage** using CSS.

---

## 📋 Requirements

- [ ] External CSS file linked properly
- [ ] Global box-sizing set to `border-box`
- [ ] Google Fonts applied (at least 1 font family)
- [ ] Color scheme with at least 3 harmonious colors
- [ ] Styled navigation bar with hover effects
- [ ] Hero/header section with gradient or background image
- [ ] Cards or sections with box shadows
- [ ] Styled form inputs with focus states
- [ ] Smooth hover transitions on links and buttons
- [ ] Proper spacing with margin and padding
- [ ] Pushed to GitHub with meaningful commits

---

## ⭐ Bonus Challenges

- [ ] Add CSS animations (keyframes)
- [ ] Create a dark/light theme toggle structure
- [ ] Add Font Awesome icons throughout the page
- [ ] Use CSS custom properties (variables) for colors
- [ ] Create a sticky navigation bar

---

## 📊 Grading Rubric

| Level | Criteria | Score |
|-------|----------|-------|
| 🟢 **Beginner** | Basic styling applied, colors and fonts used | 60-70% |
| 🟡 **Intermediate** | All requirements met, clean visual design | 70-85% |
| 🔴 **Advanced** | All requirements + bonuses, polished professional look | 85-100% |
