## 🔑 Key Concepts

- Flexbox: `display: flex`, `justify-content`, `align-items`, `flex-wrap`, `gap`
- Grid: `display: grid`, `grid-template-columns/rows`, `grid-area`, `gap`
- Flex vs Grid: when to use each
- `fr` unit, `minmax()`, `repeat()`, `auto-fit`, `auto-fill`
