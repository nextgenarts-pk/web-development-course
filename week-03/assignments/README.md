## Assignment 1: Flexbox Navigation Bar ⭐⭐

### Requirements

- [ ] Create a navigation bar with logo, menu links, and action buttons
- [ ] Logo on the left, links centered, buttons on the right
- [ ] Links have hover effects (color change + underline)
- [ ] Make it sticky at the top of the page
- [ ] On smaller screens, links should wrap naturally

### Bonus Challenges

- [ ] Add a dropdown menu on hover for one of the nav items
- [ ] Add active state styling for the current page link

---

## Assignment 2: Card Grid Layout ⭐⭐

### Requirements

- [ ] Build a grid of 6 course cards
- [ ] Each card has: image, title, description, price, and button
- [ ] Cards should be equal height with buttons aligned at the bottom
- [ ] Use `flex-wrap` so cards automatically adjust on different screen sizes
- [ ] Add hover effect: lift up + shadow increase

### Bonus Challenges

- [ ] Add a "Featured" badge on one card
- [ ] Include star ratings inside cards

---

## Assignment 3: Grid Page Layout ⭐⭐⭐

### Requirements

- [ ] Create a full page layout using CSS Grid named areas
- [ ] Include: header, sidebar, main content, and footer
- [ ] Sidebar should have a list of navigation links
- [ ] Main content should contain a grid of at least 4 stat cards
- [ ] Use media queries: single column on mobile, full layout on desktop

### Bonus Challenges

- [ ] Add a right-side panel (aside) on desktop screens
- [ ] Make the sidebar collapsible with a CSS transition

---

## Assignment 4: Responsive Portfolio — Weekly Project ⭐⭐⭐⭐

### Requirements

- [ ] **Navbar:** Sticky, Flexbox layout with logo, links, and CTA button
- [ ] **Hero Section:** Full viewport height, centered text, gradient background
- [ ] **About:** 2-column Grid layout (image + text), stacks on mobile
- [ ] **Skills:** Flexbox skill tags/badges with colors
- [ ] **Projects:** Responsive Grid with `auto-fit` + `minmax()`, minimum 4 project cards
- [ ] **Contact:** 2-column Grid (form + info), stacks on mobile
- [ ] **Footer:** Centered copyright text
- [ ] All images use `object-fit: cover` and `aspect-ratio`
- [ ] Clean typography, consistent spacing, pleasing color scheme
- [ ] Mobile-first responsive design with at least 2 breakpoints

### Bonus Challenges

- [ ] Add smooth scrolling between sections
- [ ] Include CSS animations/transitions on scroll
- [ ] Deploy to GitHub Pages
- [ ] Add a dark mode toggle (CSS only)
