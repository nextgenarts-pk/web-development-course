## Assignment 1: Responsive Blog Layout ⭐⭐

### Requirements

- [ ] Mobile-first CSS with media queries for tablet and desktop
- [ ] 1-column layout on mobile, 2-column (sidebar + content) on desktop
- [ ] Responsive navigation that changes at breakpoints
- [ ] Blog post cards that reflow from 1 to 2 to 3 columns
- [ ] All images responsive with `max-width: 100%`

---

## Assignment 2: Fluid Typography Page ⭐⭐

### Requirements

- [ ] Create a complete type scale using `clamp()` and CSS custom properties
- [ ] Build a long-form article page using the type scale
- [ ] Body text limited to `65ch` width for readability
- [ ] Fluid spacing between sections using custom properties
- [ ] Dark mode support with `prefers-color-scheme`

---

## Assignment 3: Tailwind Component Library ⭐⭐⭐

### Requirements

- [ ] Build a page showcasing these components using only Tailwind:
  - 3 button variants (primary, outline, ghost)
  - 2 card variants (product card, testimonial card)
  - Input fields with focus states
  - Alert components (success, error, warning, info)
  - Loading skeleton placeholder
- [ ] All components support dark mode
- [ ] All interactive elements have hover/focus states

---

## Assignment 4: Tailwind Portfolio — Weekly Project ⭐⭐⭐⭐

### Requirements

- [ ] Complete portfolio site using only Tailwind CSS utilities
- [ ] **Navbar:** Sticky with glassmorphism, dark mode toggle, responsive
- [ ] **Hero:** Gradient background, responsive typography, CTA buttons
- [ ] **About:** 2-column grid with profile image
- [ ] **Skills:** Colored skill badges with flex wrap
- [ ] **Projects:** Responsive grid (1→2→3 cols), hover effects, tech tags
- [ ] **Contact:** Form with styled inputs + contact info
- [ ] **Footer:** Copyright and social links
- [ ] Full dark mode support across all sections
- [ ] Smooth scroll behavior
- [ ] Deployed to GitHub Pages or Vercel

### Bonus Challenges

- [ ] Add a mobile hamburger menu with JavaScript
- [ ] Include CSS animations (fade in on scroll effect)
- [ ] Set up Tailwind CLI with a custom config
- [ ] Add a "Back to Top" button
