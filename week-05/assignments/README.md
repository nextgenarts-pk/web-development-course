## 🎯 Weekly Project: Calculator App

Build a fully functional calculator with HTML, CSS, and JavaScript.

### Requirements

- [ ] Display showing current and previous operands
- [ ] Number buttons (0-9) and decimal point
- [ ] Operator buttons (+, -, ×, ÷)
- [ ] Equals button that computes the result
- [ ] AC (clear) and DEL (delete last digit) buttons
- [ ] Proper handling of edge cases (divide by zero, multiple decimals)
- [ ] Clean, responsive CSS styling (grid layout for buttons)

### Bonus Challenges

- [ ] 🎹 **Keyboard Support:** Handle keydown events for full keyboard input
- [ ] 📜 **Calculation History:** Store and display past calculations
- [ ] 📊 **Percentage Button:** Add a % operator
- [ ] 🎨 **Theme Toggle:** Add a dark/light theme switcher
- [ ] ✨ **Animations:** Add button press animations
