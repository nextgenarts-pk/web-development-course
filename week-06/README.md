## 🔑 Key Concepts

- `document.querySelector()`, `querySelectorAll()`, `getElementById()`
- `createElement()`, `appendChild()`, `remove()`, `insertBefore()`
- `textContent`, `innerHTML`, `setAttribute()`, `classList`
- `addEventListener()`, event types (click, input, submit, keydown)
- Event object, `e.target`, `e.preventDefault()`, `e.stopPropagation()`
- Event bubbling, capturing, and delegation
- `localStorage.setItem()`, `getItem()`, `JSON.parse()`, `JSON.stringify()`
