## 🎯 Weekly Project: Interactive To-Do App

Build a fully functional To-Do application with DOM manipulation, events, and localStorage.

### Requirements

- [ ] Add new todos via form submission
- [ ] Mark todos as complete/incomplete (checkbox)
- [ ] Delete individual todos with confirmation animation
- [ ] Edit existing todos
- [ ] Filter by: All, Active, Completed
- [ ] Show count of remaining active items
- [ ] "Clear Completed" button
- [ ] Data persists in localStorage across page refreshes
- [ ] Dark/Light theme toggle with persistence
- [ ] Responsive design

### Bonus Challenges

- [ ] 🔀 **Drag & Drop** reordering of todos
- [ ] 📅 **Due Dates** with overdue highlighting
- [ ] 🏷️ **Categories/Tags** with color coding
- [ ] ✏️ **Inline Editing** (double-click to edit in place)
- [ ] ↩️ **Undo Delete** with toast notification
