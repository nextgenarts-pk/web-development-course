## 🔑 Key Concepts

- `import` / `export` (named, default, namespace)
- `class`, `constructor`, `extends`, `super`, `static`
- Private fields (`#`), getters, setters
- Enhanced destructuring patterns
- `Map`, `Set`, `WeakMap`, `WeakSet`
- `Symbol`, iterators, generators
- `try/catch/finally`, custom error classes
