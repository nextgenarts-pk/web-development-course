## 🎯 Weekly Project: OOP Quiz App

Build an interactive Quiz App using ES6 classes, modules, and modern JavaScript.

### Requirements

- [ ] `Question` class with text, options, correct answer, category
- [ ] `Quiz` class managing state (current question, answers, score)
- [ ] `QuizUI` class for all DOM rendering (separation of concerns)
- [ ] ES6 module architecture (separate files, imports/exports)
- [ ] Shuffle questions on start
- [ ] Progress bar showing completion
- [ ] Results page with score, grade, time, and answer review
- [ ] At least 10 questions across 3+ categories
- [ ] localStorage for high scores

### Bonus Challenges

- [ ] ⏱️ **Timer** — 30-second countdown per question
- [ ] 🏷️ **Category Filter** — choose which category to quiz on
- [ ] 📊 **Difficulty Levels** — easy/medium/hard with point multipliers
- [ ] 🏆 **Leaderboard** — persistent top-10 scores
- [ ] 🎨 **Animations** — slide transitions between questions
