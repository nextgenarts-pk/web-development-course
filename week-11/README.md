## 🔑 Key Concepts

- React Router v6 — `BrowserRouter`, `Routes`, `Route`
- `Link`, `NavLink`, `useNavigate`, `useLocation`
- URL parameters — `useParams`
- Search parameters — `useSearchParams`
- Nested routes and `<Outlet />`
- Shared layouts and protected routes
- Context API — `createContext`, `useContext`
- Provider pattern for global state
- Data fetching in components (`useEffect` + `fetch`)
- Loading, error, and empty states
