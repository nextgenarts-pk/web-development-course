## 🔑 Key Concepts

- Node.js event loop and non-blocking I/O
- CommonJS (`require`) vs ES Modules (`import`)
- npm, `package.json`, scripts, semantic versioning
- Express application, `app.get()`, `app.post()`, etc.
- Route parameters, query strings, request body
- Middleware pipeline — built-in, third-party, custom
- Error handling middleware
- `nodemon` for development
- HTTP methods, status codes, REST conventions
