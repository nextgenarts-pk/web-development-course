## Week Overview

This week you will connect your React frontend to your Node/Express + MongoDB backend and build real full stack workflows.

## Learning Objectives

By the end of this week, you will:

- Connect React apps to REST APIs with clean service layers
- Build complete CRUD flows across frontend and backend
- Handle authentication in the frontend using JWT/cookies
- Manage global app state for auth, loading, and errors
- Build a full stack project with protected features

## Daily Plan

1. Day 1: Full Stack Architecture and API Integration Basics
2. Day 2: Data Fetching Patterns and Service Layer Design
3. Day 3: End-to-End CRUD Flows in React + Express
4. Day 4: Frontend Authentication Integration
5. Day 5: Global State, UX States, and Error Resilience
6. Day 6: Project Day - Full Stack Notes Dashboard
